Insert Letter Tags

You can insert letter tags into the letter manually in Microsoft Word.

To insert tags in Microsoft Word 2007 and 2010:

1.   Type your generic letter in Microsoft Word 2007 or Microsoft Word 2010.

2.   Select a location in the letter where the tag should be inserted.

3.   On the Insert menu, click the Quick Parts menu, and then click Field.

The Field window opens.

4.   On the Categories menu, click Mail Merge.

5.   On the Field names menu, click MergeField.

6.   Enter the tag name in the Field Name field on the right.

7.   Click OK.

The tag is now added to the selected location in the Word document and displays in brackets <<>>.

To insert tags in Microsoft Word 2003:

1.   Type your generic letter in Microsoft Word 2003.

2.   Select a location in the letter where the tag should be inserted.

3.   On the Insert menu, click Field.

The Field window opens.

4.   On the Categories menu, click Mail Merge.

5.   On the Field names menu, click MergeField.

6.   Enter the tag name in the Field Name field on the right.

7.   Click OK.

The tag is now added to the selected location in the Word document and displays in brackets <<>>.

Note: For more information about using and inserting merge field tags in MS Word documents, refer to Microsoft Word Help.

List of Letter Tags

A list of letter tags can be accessed from the Letters module.

To access the list of letter tags:

1.   Click Patient menu on top toolbar and click Letters.

The Patient Recall window opens.

2.   Click the More (...) button in the lower left section of the window.

The Letter Templates window opens.

3.   Click Letter Tags.

The Letter Tags window opens.

4.   Click the Letter Type drop-down list and select a letter type from the following options:

•   Collection Letters

•   Encounter Forms (Superbill)

•   Progress Notes related

Letter tags related to the letter type displays.

The following sections lists the tags by the letter type. For a list of letter tags, refer to the following topics:

•   Progress Notes Related Letter Type Tags.

•   Encounter Forms (Superbill) Letter Type Tags.

•   Collection Letter Type Tags.

•   Saving the List of Letter Tags.

•   Working with Letter Tag Discrepancies.

Saving the List of Letter Tags

You can save the list of eClinicalWorks letter tags in Microsoft Excel®.

To save the list of tags in Microsoft Excel:

1.   From the Patient Hub, click Letters.

The Patient Recall window opens.

2.   Click the More (...) button.

The Letter Templates window opens.

3.   Click Letter Tags.

The Letter Tags window opens.

4.   From the Letter Type drop-down list select a letter type.

5.   Click Copy.

The Letter Tags are copied to Microsoft Excel.

You can save the Excel file for later use.

•   You can use the Find function from the Edit menu in Microsoft Excel 2003 to search for a specific tag. Alternatively, hold the Ctrl key and click F in your keyboard to use the Find function.

•   You can use the Find function from the Find & Select menu (under Home tab) in Microsoft Excel 2007 to search for a specific tag. Alternatively, hold the Ctrl key and click F in your keyboard to use the Find function.

Note: For more information about using the Find function in MS Excel, refer to Microsoft Excel Help.

Working with Letter Tag Discrepancies

A warning message displays when a letter is run with incorrect letter tags.

 

Note:  This feature is enabled by item key. Contact eCW Support and refer to the item key code 0140_UK.

This change has been implemented in the following places:

•   Patient > Letters > Run Letter

•   Progress Notes > Run Letter

•   Registry > Run Letter

•   Billing > Collection Management > Run Letter

•   Patient Hub > Letters

When the letter template is selected and run, if there are any invalid tags, then the system will prompt the following message along with the invalid tag name:

 

•   To choose not to continue, click No. Microsoft Word closes automatically.

•   Open the letter template for editing and replace the incorrect letter tag with the correct one. For more information on inserting Letter Tags, refer to Insert Letter Tags.

For more information on editing letter templates, refer to the Letters section.
